# Ordery
SQL,Flask,Python

Features
1. Full text search function
2. Upload from CSV
3. Data visualization chart


Install:
```
pip install flask

export FLASK_APP=ordery

export FLASK_ENV=development

flask init-db

python flask run --port 8000
```
